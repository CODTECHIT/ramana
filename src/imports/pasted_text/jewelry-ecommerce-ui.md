# Figma Make Prompt — Premium Jewelry E-Commerce UI

Paste this directly into Figma Make. This is UI/UX-focused only — no backend, no real data, just a premium, realistic-looking design.

---

Design a premium, high-end jewelry e-commerce website UI for an Indian gold and bridal jewellery brand. The aesthetic should feel like a luxury heritage jewellery house — think Pothys Swarna Mahal and Ramana Jewells — NOT like a generic Shopify template or a typical SaaS product. It should feel expensive, editorial, and considered.

## Visual Style

- **Color palette:** warm ivory/cream background (not stark white), deep antique gold (#C9A227 / #B8860B) as the primary accent for buttons, borders, dividers and hover states, deep maroon (#5C1A1B) as a secondary jewel-tone accent, warm charcoal (#241F1A) for text — never pure black, never pure white
- **Typography:** an elegant high-contrast serif (like Playfair Display or Cormorant Garamond) for all headings and product names, paired with a clean sans-serif (like Inter) for body text, prices, and UI labels. Use generous letter-spacing and small uppercase text for navigation and category labels
- **Overall feel:** generous whitespace, large confident hero typography, restrained gold used as punctuation (borders, icons, dividers) rather than flooding backgrounds, soft drop shadows on product cards, rounded-but-refined corners (not overly rounded/playful)
- Avoid: purple/blue SaaS gradients, flat white backgrounds, generic icon-only UI with no personality, anything that looks like a free template

## Pages to design (as separate frames/screens)

### 1. Homepage
- Top utility bar showing a live gold rate ticker (e.g. "Gold 22KT — ₹13,375/g") and currency selector
- Header with logo (centered or left), navigation for categories: Necklaces, Long Harams, Hipbelts, Tikkas, Hair Accessories, Bangles, Bridal Sets — plus search icon, account icon, wishlist icon, cart icon
- Full-bleed hero banner with a bridal jewellery model image and large serif headline overlay + gold CTA button
- "Shop by Category" section — circular/arched image tiles (not square cards) for each of the 7 categories, with serif labels underneath
- "Collections" editorial section — 2-3 large lifestyle photography blocks (e.g. "The Bridal Edit", "Temple Jewellery", "Everyday Gold") with title + short description + "Explore" link
- Bestsellers product carousel — product cards with image, name, price, small wishlist heart icon
- Trust badge row — BIS Hallmark Certified, Lifetime Exchange, Secure Payments, Easy Returns (small icons + labels)
- Newsletter signup band with gold accent background
- Footer with link columns (About, Contact, FAQ, Shipping & Returns), social icons, WhatsApp contact button

### 2. Category Listing Page (e.g. "Bridal Sets")
- Breadcrumb (Home / Bridal Sets)
- Left sidebar filters: price range slider, weight range, occasion tags, sort dropdown
- Product grid (3-4 columns desktop) — each card shows product image, hover state showing a second angle, product name, price, small "New" or "Bestseller" tag where relevant
- Pagination at the bottom

### 3. Product Detail Page
- Left: large image gallery with thumbnail strip, zoom-on-hover interaction
- Right: product name (serif, large), price, short description, variant selector (e.g. bangle size or chain length as pill-shaped selectable buttons), "Set Contents" expandable accordion for bridal sets showing included pieces, quantity selector, "Add to Cart" gold button, "Enquire on WhatsApp" secondary button
- Below: tabs or accordion for Product Details, Care Instructions, Shipping & Returns
- Bottom: "You may also like" related products carousel

### 4. Cart / Checkout
- Step indicator (Cart → Address → Payment)
- Cart: line items with image, name, variant, price, quantity stepper, remove icon; order summary card on the right with subtotal, shipping, total, and gold "Proceed to Checkout" button
- Checkout: address form, saved addresses list, payment method selection (Razorpay options + COD), final order summary

### 5. Admin Dashboard (separate, simpler visual style)
- Clean sidebar navigation: Dashboard, Products, Categories, Orders, Customers, Banners, Reports
- Dashboard home: revenue chart (line or bar), key metric cards (Total Orders, Revenue This Month, Low Stock Items, New Customers), recent orders table
- Product list view: data table with thumbnail, name, category, stock, price, status toggle, edit/delete icons, "Add Product" button top right
- This screen can use a more neutral, functional design system (whites, grays, single accent color) — it doesn't need the same heavy brand treatment as the storefront, just clean and professional

## Interactions to imply in the design
- Hover states on product cards (image swap or subtle lift + shadow)
- Active/selected states on variant pills and filters
- Gold underline or border animation cue on nav items (note as an annotation if Figma Make doesn't render motion)

## Output
Generate this as a connected multi-screen prototype with consistent components (buttons, product cards, nav, footer) reused across screens so it feels like one cohesive design system, not disconnected mockups.
